# Онбординг-плафторма "Компас"

## Стек
- PHP >= 7
- YII2
- Vue.js
- MySQL
- Nginx
- Ubuntu Server 20.04

## Установка

````
1) composer install
2) cp .env.example .env
3) Прописать настройки подключения к БД
4) php yii migrate
````
